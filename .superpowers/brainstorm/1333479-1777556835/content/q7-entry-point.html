User vào Editor Mode từ đâu?

Flow để user đi từ giao diện hiện tại vào editor mode

A

Từ Dashboard — "Open in Editor" button

Dashboard hiện tại group session theo Machine → Directory. Mỗi group có thêm nút "Open in Editor". Click → navigate sang /editor?machine=...&project=....

B

Từ Session — button "Open in Editor"

Trong session detail, có nút "Open in Editor" trên header. Tự động lấy machine từ session metadata + project path.

C

Cả A và B

Cả 2 entry point. Dashboard cho "bắt đầu mới", session cho "đang chat muốn xem code".
