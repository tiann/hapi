Chọn project trong Editor Mode như thế nào?

Vì không có "project" concept chính thức, cần xác định cách user chọn workspace

A

Url param + browse từ header

URL: /editor?project=/home/user/foo. Trên header có dropdown chọn project — browse machine + folder giống WorkspaceBrowser. Đổi project → load lại tree + list session của project đó.

B

Không chọn project — workspace là tất cả session

Editor mode hiển thị tất cả session từ tất cả project. File tree là "merged view" từ nhiều session. Phức tạp nhưng không cần concept project.

C

Tự động từ session active

Khi click "Open in Editor" từ 1 session đang chạy, tự động lấy metadata.path làm project root. Không cần chọn thủ công. Header vẫn có thể đổi sau.
