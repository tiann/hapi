# 渐进式 Web 应用（PWA）

主神的 Web 界面是一个功能完整的 PWA，可安装到手机上，获得接近原生应用的使用体验。

## 什么是 PWA？

渐进式 Web 应用（PWA）是一种可安装到设备上的 Web 应用，使用方式类似原生 App：

- **主屏图标**：像普通应用一样从桌面启动主神
- **全屏模式**：无浏览器地址栏，体验更沉浸
- **离线支持**：无网络时仍可使用基础功能
- **自动更新**：始终获取最新版本

## 安装 HAPI PWA

### Android（Chrome/Edge）

1. 在 Chrome 或 Edge 中打开主神
2. 在底部找到 **”Install HAPI”** 横幅
3. 点击 **”Install”**
4. 主神会出现在主屏幕

::: tip
如果没有看到安装横幅，可点击右上角菜单，选择 **“Add to Home screen”** 或 **“Install app”**。
:::

### iOS（Safari）

1. 在 Safari 中打开主神
2. 点击 **分享** 按钮（方框上箭头）
3. 下滑并点击 **“Add to Home Screen”**
4. 点击右上角 **“Add”**

::: warning
iOS 仅支持通过 Safari 安装 PWA。iOS 上的 Chrome/Firefox 不支持“Add to Home Screen”。
:::

### 桌面端（Chrome/Edge）

1. 在浏览器中打开主神
2. 点击地址栏中的安装图标（⊕）
3. 或使用菜单：**“Install HAPI...”**
4. 主神将以独立窗口打开

## PWA 功能

### 离线模式

离线时，主神可以：

- 展示缓存的会话列表
- 显示已加载过的消息
- 将操作排队，待网络恢复后执行

断网时会显示离线状态提示。

### 自动更新

主神会自动检查更新：

- 每小时后台检查一次
- 有新版本时会提示
- 点击 “Reload” 即可更新到最新版本

### 后台同步

离线期间产生的操作会在重连后同步：

- 发送待处理消息
- 同步权限决策
- 刷新会话状态

## 缓存策略

主神采用智能缓存策略：

| 内容 | 策略 | 时长 |
|---------|----------|----------|
| App shell | Cache first | 直到更新 |
| Sessions API | Network first | 5 分钟 |
| Machines API | Network first | 10 分钟 |
| 静态资源 | Cache first | 永久 |

## 通知

当 Agent 需要你处理时，主神支持推送通知提醒。

### 启用通知

1. 打开主神后会自动弹出权限请求
2. 点击 **Allow** 启用通知
3. 若错过弹窗，可到系统设置手动授予权限

### 通知类型

| 类型 | 触发时机 |
|------|-----------|
| Permission Request | Agent 需要你审批 |
| Ready | Agent 完成并等待输入 |

::: tip
如果你所在地区推送不可用（如 FCM 不可达），可考虑自托管通知中继服务。
:::

## 管理你的 PWA

### 查看安装状态

主神会根据安装状态显示不同 UI：

- **未安装**：显示安装提示
- **安装中**：显示进度状态
- **已安装**：不再显示安装提示

### 卸载

**Android：**
1. 长按 HAPI 图标
2. 拖动到“卸载”或点击 X

**iOS：**
1. 长按 HAPI 图标
2. 点击 “Remove App” → “Delete App”

**桌面端：**
1. 打开 HAPI
2. 点击右上角三点菜单
3. 选择 “Uninstall HAPI”

### 清理缓存

如遇异常可尝试：

1. 在浏览器中打开主神（不是安装版）
2. 打开开发者工具（F12）
3. 进入 Application → Storage
4. 点击 “Clear site data”

## 最佳实践

### 电池优化

在 Android 上，建议为主神关闭电池优化，以保证：

- 后台同步稳定执行
- 通知及时到达

路径：Settings → Apps → HAPI → Battery → Unrestricted

### 流量使用

主神流量占用较低：

- 首次加载约 ~500KB
- 首次后资源会缓存
- 仅同步发生变更的数据

### 多设备使用

你可以在多个设备安装主神：

- 所有设备连接同一个服务端
- 会话可跨设备同步
- 同一 access token 可在各设备使用

## 故障排查

### 看不到安装按钮

- 确认使用 HTTPS（PWA 必需）
- 尝试刷新页面
- 检查是否已经安装

### 应用不更新

1. 完全关闭应用
2. 重新打开并等待更新提示
3. 若仍异常，清缓存并重新安装

### 离线模式不可用

- 确认至少曾在线打开过一次应用
- 检查 ServiceWorker 是否注册（DevTools → Application）
- 清缓存后重新加载

### iOS 特有限制

- 必须使用 Safari 安装
- iOS 不支持后台同步
- 离线能力相对受限
